# Проект 4: Место

### Блоки
* header - здесь содержится только логотип и нижнее подчеркивание блока
* main - секция разметки основного контента
  * profile - секция с подменяемыми данными имени и описания через всплывающую форму, которая открывается при нажатии на кнопку редактирования
  * grid - секция с портфолио: фото, описание и возможность поставить лайк
* footer - копирайт

* popup (диалоговое окно) - находится в конце документа html, скрыто по умолчанию, открывается только при нажатии на кнопку редактирования из секции prolile, также затемняет фон всего сайта и содержит подменяемые данные этой же секции.
